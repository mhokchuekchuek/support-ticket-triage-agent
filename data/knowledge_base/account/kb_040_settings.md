## Accessing Settings

Click your profile avatar in the top-right corner and select **Settings**, or use the keyboard shortcut `Ctrl/Cmd + ,`.

## Profile Settings

### Personal Information
- **Display name**: How others see you
- **Username**: Your unique identifier (@username)
- **Email**: Primary contact email
- **Phone**: Optional, for 2FA backup
- **Profile photo**: Upload or use Gravatar

### Updating Your Email
1. Go to Settings > Profile > Email
2. Enter new email address
3. Verify via confirmation link sent to new email
4. Old email notified of change for security

## Notification Settings

### Email Notifications
Configure what triggers email alerts:
- **Activity**: Comments, mentions, assignments
- **Updates**: Project changes, status updates
- **Digest**: Daily or weekly summary
- **Marketing**: Product news (can opt out)

### In-App Notifications
- Real-time popup notifications
- Notification center (bell icon)
- Configure sound alerts

### Mobile Push Notifications
- Requires mobile app installation
- Mirror email notification settings
- Or configure separately

### Do Not Disturb
- Schedule quiet hours
- Pause all notifications temporarily
- Set status to away

## Preferences

### Language
- Interface language selection
- 20+ languages supported
- Affects UI only, not content

### Timezone
- Auto-detect from browser
- Manual override available
- Affects all date/time displays
- Important for scheduled notifications

### Date/Time Format
- 12-hour or 24-hour clock
- Date format (MM/DD/YYYY, DD/MM/YYYY, etc.)

### Keyboard Shortcuts
- View all shortcuts: `?` key
- Enable/disable shortcuts
- Customize common actions (Pro)

## Workspace Settings

If you manage a workspace:

### General
- Workspace name and description
- Workspace icon
- Default timezone for workspace

### Members
- View all members
- Invite new members
- Manage roles and permissions
- Remove members

### Security
- Require 2FA for all members
- Session timeout settings
- IP allowlist (Enterprise)

## Connected Accounts

Link external accounts:
- Google (for SSO and Drive)
- GitHub (for repository linking)
- Slack (for notifications)
- Microsoft (for Teams integration)

### Managing Connections
1. Settings > Connected Accounts
2. View all linked accounts
3. Disconnect accounts you no longer use
4. Permissions shown for each connection

## Data & Privacy

### Export Your Data
- Download all your personal data
- Available once per 30 days
- GDPR compliant

### Delete Account
1. Settings > Account > Delete Account
2. Enter password to confirm
3. 30-day grace period before permanent deletion
4. Data exported option before deletion

### Privacy Settings
- Profile visibility (public/private)
- Activity status visibility
- Search engine indexing (public profiles)
