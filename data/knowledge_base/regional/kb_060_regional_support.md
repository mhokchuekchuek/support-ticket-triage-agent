## Data Residency

### Default Data Location
- Free/Pro: US-East by default
- Can request region during signup (Pro)
- Enterprise: Choose any available region

### Changing Regions
- Contact support to migrate data
- Migration takes 24-48 hours
- Brief read-only period during migration
- Enterprise: Dedicated migration support

### Compliance Considerations
- **GDPR**: EU data centers available
- **Data sovereignty**: Region-specific hosting
- **Custom requirements**: Contact Enterprise sales

## Language Support

### Interface Languages

Currently supported:
- English (default)
- Spanish (Español)
- French (Français)
- German (Deutsch)
- Portuguese (Português)
- Japanese (日本語)
- Korean (한국어)
- Chinese Simplified (简体中文)
- Chinese Traditional (繁體中文)

### Thai Language Status

**Current Status**: Not yet available

**Roadmap**:
- Thai language support is on our roadmap for Q3 next year
- Priority based on user demand
- Vote for Thai support at feedback.example.com

**Workarounds**:
- Content can be in Thai (Unicode supported)
- Interface remains in English or other supported language
- Browser translation extensions work with our app

### Requesting New Languages
1. Visit feedback.example.com
2. Search for your language request
3. Vote if exists, or create new request
4. Include use case and user count estimate

## Timezone Support

### Setting Your Timezone
1. Go to **Settings > Preferences > Timezone**
2. Choose your timezone from dropdown
3. Or enable "Auto-detect from browser"

### Timezone Features
- All timestamps display in your local time
- Scheduled notifications respect timezone
- Team members see times in their own zones
- Meeting scheduler shows timezone conflicts

### Workspace Timezone
Admins can set workspace default:
- Settings > Workspace > Default Timezone
- Affects new members and shared views
- Individual settings override workspace default

## Localization Limitations

### Currently Not Localized
- API responses (English only)
- System emails (English, with translation roadmap)
- Documentation (English primary)
- Support chat (English, with translation available)

### Unicode Support
- Full UTF-8 support for content
- Thai, Arabic, Hebrew script supported
- RTL languages supported in content
- File names support all characters
